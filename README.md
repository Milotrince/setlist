# setlist

simple tabs & chord sheets. fork and host your own!

song files located in [src/lib/songs)](src/lib/songs)


## features
- [x] multicolumn view
- [x] autoscroll
- [x] text resize
- [x] chord fingerings for guitar
- [x] transpose
- [x] light/dark mode
- [x] search
- [x] tags
- [ ] ultimateguitar search
- [ ] robust chords
